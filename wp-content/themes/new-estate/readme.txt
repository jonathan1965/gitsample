### How to set static homepage like screenshot ###
==================================
Go to WordPress admin => pages => Add new page => select template 'HomePage' and publish the page.
Then go to customizer => static front page => check 'A static page' => chose the above saved page from dropdown list.
Then go to customizer => Home page settings => Edit settings as you wish. That's all.
